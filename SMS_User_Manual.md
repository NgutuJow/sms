# School Management System User Manual

## 1. Introduction
This user manual explains how to use the School Management System installed in this project.
It is designed for administrative staff, teachers, and parents to manage school operations, academic records, finance, attendance, and reports.

## 2. System Overview
This system supports:
- user login and role-based access
- school, branch, teacher, and student management
- academic setup for sessions, semesters, classes, streams, subjects, timetables, and syllabus
- attendance tracking and review
- exam and mark management for teachers and administrators
- invoices, payments, receipts, fines, budgets, expenses, payroll, and financial reports
- student reports, exam reports, attendance reports, and PDF export
- announcements and audit logging

## 3. Accessing the System
### 3.1 Login
- Open the application URL in your browser.
- Use the login page to enter your email and password.
- Successful login redirects to the dashboard.

### 3.2 Password Recovery
- Click "Forgot Password" on the login page.
- Enter your registered email.
- Follow the reset link in the email.
- Set a new password with confirmation.

### 3.3 Change Password
- Once logged in, go to "Change Password".
- Enter your current password, new password, and new password confirmation.
- Save the update.

## 4. User Roles and Permissions
### 4.1 Administrator
Administrators have full access to:
- user management
- academic setup
- finance management
- reports and exports
- master data configuration

### 4.2 Teacher
Teachers can access:
- exam marking and progress
- exam reports
- teacher-specific attendance and review pages
- teacher resources

### 4.3 Parent
Parents can access:
- student attendance
- invoices and fee statements
- receipts and payments history
- exam reports

### 4.4 Student
Students have restricted access to their own academic and financial information.

## 5. Dashboard Overview
After login, the dashboard displays key school metrics:
- number of branches, students, teachers, classes
- admissions and attendance statistics
- sums collected and pending fees
- latest exams, announcements, and recent activities

Use the dashboard to quickly check school health and recent updates.

## 6. Managing Schools and Branches
### 6.1 Create a School
- Go to the school management section.
- Click "Create School".
- Enter school name, address, and details.
- Save the record.

### 6.2 Branch Management
- Access branches from a selected school.
- Create a new branch with branch name, location, and status.
- Edit branch details by selecting the branch and updating values.
- Toggle branch status to activate or deactivate.
- Delete branches when they are no longer required.

## 7. Teacher Management
- Open the Teachers section.
- Create a teacher record with name, contact details, and assigned branch.
- Update teacher details as needed.
- Toggle teacher status to manage active and inactive staff.

## 8. Academic Setup
### 8.1 Sessions and Semesters
- Create academic sessions for each school year.
- Create semesters inside academic sessions.
- Set an active session and semester to use across attendance, exams, and student records.
- Delete outdated sessions or semesters when necessary.

### 8.2 Classes and Streams
- Create school classes (grade levels) as needed.
- Create streams to group students inside a class.
- Assign stream names and link them to classes.
- Remove unused classes and streams.

### 8.3 Subjects
- Add subjects to the academic structure.
- Use subjects in examination and mark entry pages.
- Maintain subject listings for each class or stream.

## 9. Student Management
### 9.1 Add Students
- Go to the student management section.
- Click "Add Student" and fill in admission number, name, class, stream, session, semester, parent details, and contact information.
- Save the student profile.

### 9.2 Edit Students
- Search or browse the student list.
- Select a student to update details like class, branch, or contact information.
- Save the changes.

### 9.3 Bulk Upload
- Use the bulk import page to upload many student records at once.
- Follow the template format if available.
- Validate imported records and correct any errors.

## 10. Attendance Management
### 10.1 Record Attendance
- Go to the Attendance page under Academic or Teacher menus.
- Select class, stream, and date.
- Mark each student as present, absent, or another status.
- Save attendance data.

### 10.2 Review Attendance
- Access the attendance review page to verify recorded entries.
- Search by date, class, or stream.
- Correct records using the editing or delete options.

### 10.3 Attendance Reports
- Generate attendance reports to view student attendance history.
- Download attendance PDFs from the reports page.
- Use attendance reports for parent or administrator review.

## 11. Timetable and Syllabus
### 11.1 Timetable
- Open the timetable section.
- Create schedule entries for days, subjects, teachers, and class periods.
- Update or delete timetable entries as needed.

### 11.2 Syllabus
- Create syllabus entries for classes and subjects.
- Use the syllabus module to store academic content or notes.

## 12. Examination and Mark Management
### 12.1 Teacher Exam Marking
- Log in as a teacher and open the teacher exam dashboard.
- Select an exam to view the list of assigned students.
- Enter marks for each student.
- Save all entries or use the single-mark save for quick updates.
- Upload marks in bulk using the bulk upload feature and a download template when available.

### 12.2 Exam Progress and Reports
- View progress pages for exam completion status.
- Generate teacher-level reports and exam results.
- Download PDF versions of exam reports if needed.

### 12.3 Admin Exam Management
- As an administrator, create, edit, and delete exam entries.
- View exam reports and results overview charts.
- Export results to PDF for record keeping.
- Generate student performance reports for the whole school or individual students.

## 13. Finance and Fees
### 13.1 Fee Structures
- Define fee structures for classes and student categories.
- Add fee items and link them to classes or academic sessions.
- Update fee amounts as needed.

### 13.2 Invoices
- Create invoices for students based on fee structures.
- Review invoice details, due amounts, and status.
- Generate invoices for new admissions or current fees.

### 13.3 Payments
- Record payments against invoices.
- Enter payment amount, method, and student reference.
- Track paid and unpaid invoices.

### 13.4 Receipts
- View receipts for completed payments.
- Download or print receipts for parents and record keeping.

### 13.5 Discounts and Fines
- Apply discounts to student fees when authorized.
- Record and manage fines for late payments or penalties.
- Review fine details and update status.

### 13.6 Expenses and Budgets
- Record school expenses with descriptions and amounts.
- Manage budgets by category and track spending against budget limits.
- Use financial reports to monitor expenses.

### 13.7 Payroll
- Create payroll records for school staff.
- Enter salary details, allowances, and deductions.
- Generate payroll reports and manage payroll history.

## 14. Reports and Exports
### 14.1 PDF Export
- Use the export features in finance, exams, attendance, and reports pages.
- Download year-end reports, payroll reports, expense reports, and financial summaries in PDF format.

### 14.2 Student Reports
- Generate student-specific academic reports.
- Download and share student report PDFs with parents.

### 14.3 Exam Results and Overview
- Access exam results overview pages.
- Generate charts and summary reports for exam performance.
- Export exam result PDFs when needed.

## 15. Announcements and Chat
- Create announcements to share information with staff, students, or parents.
- Use the chat module to send messages and coordinate school activities.
- Review announcements from the dashboard.

## 16. Audit Logs and Backup
### 16.1 Audit Logs
- Review audit logs to track user activity and system events.
- Use logs for security review and change tracking.

### 16.2 Backup
- Access the system backup page.
- Create or restore backups for critical data.
- Follow the backup process carefully.

## 17. Parent Portal
- Parents can view their child’s attendance, invoices, receipts, and exam reports.
- Use the parent dashboard to navigate financial and academic records.
- Monitor payment history and fee statements.

## 18. Teacher Resources
- Teachers can access lesson materials and resources.
- The resources section may include files, links, or syllabus content.

## 19. Best Practices
- Keep user accounts updated and remove inactive users.
- Maintain accurate academic sessions, semesters, and class assignments.
- Record attendance daily and review reports regularly.
- Reconcile payments, invoices, and receipts frequently.
- Run backups often and review audit logs for unusual activity.

## 20. Troubleshooting
- If a page fails to load, refresh and check user permissions.
- If login fails, use "Forgot Password" or contact the administrator.
- For missing records, verify the active academic session and semester.
- Use the dashboard metrics to identify data gaps.

## 21. Glossary
- Dashboard: The home screen with school metrics and quick access links.
- Session: The academic year or period.
- Semester: A division of the academic year.
- Stream: A group inside a class.
- Invoice: A bill generated for student fees.
- Receipt: Proof of payment.
- Audit Log: Record of system actions and changes.

## 22. Notes
- This manual is based on the core modules available in the system.
- Some labels and menu names may vary depending on deployment and customization.
- Administrators should update this manual when new features are added.
